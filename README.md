## Map Tiles Lab


In this project The geographic area examined in this project is Seattle, Washington. This web map can be found [here.](https://milaboj.github.io/mapTiles/) 


The first tile set is a simple base map with an orange color scheme with a bit of green as seen below.

![base](/img/base.png)

The second tile set shows the areas in Seattle that are creek watersheds which are piped to outfall seen below. Data source: [Seattle GeoData](https://data-seattlecitygis.opendata.arcgis.com/datasets/SeattleCityGIS::piped-creek-watersheds/about) 

![water](/img/watershed.png)

The third tile set combines the previous tile sets into one where both layers are visible at once.

![comb](/img/combined.png)

The last tile set 

![4](/img/combined.png)


The base map tile set, piped creek watershed tile set, combined tile set, and mapbox tile set each have zoom levels available from 0-10.
